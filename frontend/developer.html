🪄 MVP: Decentralized Database on NEAR

👩‍💻 Developers

Write Database Schema and ETL logic code and publish them on-chain using a custom DB3 smart contract in which only the CID of the bundle is stored.

🌐 Hosts

Choose which databases you're interested to host. Pay a security deposit and register their API endpoints on-chain. Utilize DB3 nodes to launch and sync databases.

👤 Users

Discover API endpoints for databases you're interested in. Sign queries with an attached fee payments and send them to selected hosts for execution.
